# AIBR - All-Inclusive Resort

A polished HTML/CSS landing page for a luxury adult all-inclusive resort called **Sanctuary Cap Cana**. The page showcases a resort hero gallery, resort details, navigation, and responsive styling.

## Project Structure

- `index.html` - Main landing page
- `css/styles.css` - Styling for layout, typography, and responsive design
- `assets/` - Images and logo assets used in the design

## Features

- Clean resort landing page layout
- Header navigation, branding, and contact elements
- Breadcrumb navigation and section headings
- Image hero section with gallery preview button
- Resort details including rating, bedroom/bathroom count, and guest capacity
- Google Fonts integration and Font Awesome icons

## How to View

Open `index.html` in your browser to preview the page.
